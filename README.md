# spaceEasel

Easel.js + Meteor

An attempt at making Easel.js work with Meteor in a Meteor way.

Features : 

* Reactive Canvas Elements
* Reactive Canvas Text
* Persistent and non persistent canvas entities

![Click and drag](http://i.imgur.com/FuEczzB.gif)
